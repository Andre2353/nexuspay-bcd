# nexus-pay

sistema de gestão de pagamentos e infoprodutos.

## o que faz
- processa vendas (pix e cartão).
- gerencia comissões de afiliados.
- controla o saldo e saques dos usuários.

## arquivos
- `script.sql`: tabelas de usuários, produtos e vendas.
- `diagrama.png`: modelo do banco de dados.

projeto para estudo de fluxos financeiros.
